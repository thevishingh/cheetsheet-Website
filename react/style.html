React Css Code Snippits

    React CSS Styling

    There are three ways to style in React

  1. → Inline Css
  2. → Css Stylesheet
  3. → Css Modules

    4. Inline Styling

    To inline style an element, we can make a Javascript Object, like this 

  4. 
const App = () => {
  return (
    <>
      <h1 style={{ backgroundColor: "purple" }}>CodeWithHarry</h1>
    </>
  );
}

    5. In this first curly brace is to write javascript and second is to make a javascript object. We can also write it like 

  5. 
const App = () => {
const h1Style = {
 backgroundColor: 'purple',
 color: 'white'
}
return (
 <>
 <h1 style={h1Style}>CodeWithHarry</h1>
 </>
);
}

    6. Note: CSS property name must be camelCase. background-color would be backgroundColor

    CSS Stylesheets

    You can save the whole CSS in a separate file with file extension .css and import it in your application App.css 

  6. 
body {
  background-color: 'purple';
  color: 'white';
}

    7. Here we are writing CSS, so we don't need to make JS Object or do it in camelCase
    Just import it like this → Index.js 

  7. 
import './App.css'

    8. CSS Modules

    In this you don't have to worry about name conflicts as it is component specific. The CSS is available only for the file in which it is imported

  8. → Make a file with extension .module.css, example: index.module.css
  9. → Make a new file index.module.css and insert some CSS into it, like this:
  10. 
.button {
background-color: 'purple';
color: 'white';
}

    11. Import it in component like this 

  11. 
const Home = () => {
   return (
     <button className={styles.button}>Click me!</button>
     )
 };
export default Home;
