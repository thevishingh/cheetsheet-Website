React Conditional Code Snippits

    React Conditional

    One of the best thing in React is in React we can conditionally render elements/components!

    && Operator

    This is one of the way to conditionally render 

  1. 
function App() {
const loading = true;
return (
    <div& className="App">
      {loading && <LoadingGif />}
      {!loading && <Content />}
    </div>
  );
}

    2. Here we are saying if loading is true then show → <LoadingGif /> and in second line, we said !loading so that returns false in this case so → <content /> will not load and first statement is true so → <LoadingGif /> will load!

    Ternary Operator

    What we were doing in previous example was basically just if statement. We don't have else so we were using if as else too. Let's learn how to do both (if/else)! Same thing can be done like this: 

  2. 
function App() {
const loading = true;
return (
    <div& className="App">
      {loading ? <LoadingGif /> : <Content />}
    </div>
  );
}

    3. We are saying if loading is true then show → <LoadingGif />
    else show → <content /> simple!

    Syntax

  3. 
condition ? true : false
